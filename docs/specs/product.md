# PoseFinder: Product（プロダクト仕様）

- **最終更新日**: 2026-01-09
- **正本**: `docs/specs/*`（`docs/archive/specs/*` は参考資料）

## 1. 背景 / 目的

PoseFinder は、筋力トレーニング中のフォームを **客観的に記録・確認**できるようにする iOS アプリです。  
ユーザー本人のカメラ映像（Video）と姿勢（Pose）情報を同時に保存し、後から同期して再生・振り返りできる状態を目指します。

## 2. 想定ユーザー / 利用シーン

- 自宅やジムでフォーム改善を目指す個人
- トレーニングの再現性を高めたい人（前回との差分を振り返りたい）
- コーチングを受けている人（「手本」との比較材料を残したい）

## 3. 主要ユーザージャーニー（MVP）

### 3.1 Recording（撮影して保存）

1. ホームからトレーニングメニューを選ぶ
2. 詳細画面で「撮影を開始」
3. 撮影画面で（現状は手本動画の再生と同時に）録画・Pose 検出・保存が進む
4. 撮影終了でセッションが保存される

### 3.2 Session Replay（セッションを再生して確認）

1. ホームの「履歴」からセッション一覧を開く
2. セッションを選択して詳細画面へ
3. 動画再生と Pose 同期オーバーレイが確認できる

## 4. スコープ

### 4.1 MVP に含めるもの

- セッションのローカル保存（Video + Pose NDJSON + session.json）
- セッション一覧表示（保存済みセッションの列挙）
- セッション詳細表示（動画再生 + Pose 同期オーバーレイ + メタ情報表示）
- 中断時のセッション破棄（保存されないことが明確であること）

### 4.2 非目標（現時点）

- クラウド同期 / 複数端末共有
- 本格的なフォーム採点（総合スコアの安定化・フィードバック提示）
- 解析結果の編集・注釈・共有（SNS 等）
- DB 導入（現状はファイルベース保存）
- 長尺・高fpsセッションでの効率的な Pose 再生最適化

## 5. 用語

- **Session**: 1回のトレーニング記録（`video.mp4` + `pose.ndjson` + `session.json`）
- **Video**: ユーザー本人を撮影した録画データ
- **Movie**: 手本となる事前収録の模範動画（現状はバンドル動画）
- **Pose**: 推定された人体の関節位置（推論結果）
- **NDJSON**: 1行=1フレームの JSON Lines 形式（`pose.ndjson`）

## 6. 成功指標（当面の目安）

- 撮影 → 保存 → 一覧 → 詳細表示までが、特別な操作なしに完遂できる
- 中断時に壊れたデータが一覧に残らない（または「不完全」を明示して選べない）
- 端末ローカルでの保存・再生が安定する（一定時間の録画でクラッシュしない）
