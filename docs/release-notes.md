# Release Notes

## 2026-03-18

### 0010: セッション再生最適化

- `pose.ndjson` の再生経路を、全件配列化から「時刻インデックス + オンデマンド読み込み」方式へ変更し、長尺セッションでの表示安定性を改善。
- `SessionDetailViewModel` に近傍フレーム先読みとサイズ上限制御付きキャッシュを追加し、再生同期時のメモリ増加を抑制。
- 実機回帰で見つかった起動遅延とお手本動画前半のカクつきに対し、起動時リソースコピーの非同期化/差分コピー化、および再生バッファ設定を追加。
- `PoseFinderTests` に長尺データ向けのインデックス生成・オンデマンド読み込み・改善率検証テストを追加。

## 2026-03-12

### 0009: MVP 完全化ステップ1

- `PoseFinderTests` / `PoseFinderUITests` を整備し、撮影完了遷移と Pose 同期表示の自動テストを追加。
- UI テスト用のシードデータ投入と `accessibilityIdentifier` ベースの判定を導入し、回帰確認を安定化。
- セッション詳細での Pose 同期オーバーレイ仕様を `docs/specs/ui.md` と `docs/specs/tech.md` に反映。
- `main` に fast-forward マージし、リモートへ反映済み。
